SFSUpdate by davromaniak

Updates the banlist and disallow tables of common forum engines with the http://www.stopforumspam.com/ banlist

This script downloads files from http://www.stopforumspam.com, does some treatment, inserts it in the database, remove the duplicate entries, and that's all

Dependencies

  • - bash
  • - wget
  • - grep
  • - sed
  • - mysql client and server
  • - zcat
  • - getopt
  • - date
  • - eval
  • - gawk
  • - and of course, a phpbb database installed.

Install

- Give the correct rights in order to execute the sfs_update script (usually a "chmod u+x sfs_update" should do the trick)
